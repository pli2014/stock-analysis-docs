**15 项检查清单**（强制执行）:
```yaml
估值维度:
  - PE < 25x ✓
  - PB < 3x ✓
  - 当前价/内在价值 < 0.7 ✓

财务健康:
  - ROE > 15% (连续 3 年) ✓
  - 营收增长 CAGR > 20% ✓
  - FCF/净利润 > 0.7 ✓
  - 资产负债率 < 60% ✓

分红政策:
  - 连续 3 年分红 ✓
  - 股息率 > 2% ✓

治理质量:
  - 审计意见为标准无保留 ✓
  - 商誉占比 < 30% ✓
  - 高管持股 > 5% ✓
  - 近期无大股东减持 ✓
  - 关联交易 < 10% ✓

护城河与行业:
  - 护城河类型清晰 ✓
  - 管理层诚信可靠 ✓
  - 行业竞争格局良性 ✓
  - 无重大监管风险 ✓
```

### Layer 3: Daily Operation (日常运营层)

#### 定时任务配置 (`cron/jobs.json`)

**执行时间**: 工作日 17:00 (Asia/Shanghai)  
**下次运行**: 2026-04-13 17:00:00 (周一)

**工作流程图**:
```
┌──────────────────────────────────────────────────────────┐
│              DAILY MARKET ANALYSIS PIPELINE              │
└──────────────────────────────────────────────────────────┘

Start (17:00)
    │
    ├→ Step 1: 基本面初筛 
    │   ├─ 排除 ST/*ST、PE>35x、ROE<12%、非标准审计
    │   ├─ 筛选 ROE>15%、FCF/净利润>0.7、连续分红
    │   └─ Output: daily-reports/YYYY-MM-DD-初筛清单.md
    │
    ├→ Step 2: 深度研究
    │   ├─ 商业模式评估 (护城河/颠覆风险/转换成本)
    │   ├─ 管理层审查 (增持减持/股权激励/关联交易)
    │   ├─ 财务健康度 (现金流/周转率/毛利率趋势)
    │   ├─ 行业竞争格局 (市占率/增速匹配/监管风险)
    │   └─ Output: thematic-reports/YYYY-MM-DD-深度研究.md
    │
    ├→ Step 3: 持仓监控
    │   ├─ 公告扫描 (重大合同/高管变动/业绩预告)
    │   └─ 卖出信号检查 (逻辑证伪/财务造假/业务颠覆)
    │
    ├→ Step 4: 机会池更新
    │   ├─ 添加新标的 / 移除不合格标的
    │   └─ Output: learning-notes/YYYY-MM-DD-观察池更新.md
    │
    ├→ Step 5: 量价形态辅助 ⭐
    │   ├─ 判断四种量价模型 (增升/缩升/增跌/缩跌)
    │   └─ Output: daily-reports/YYYY-MM-DD-量价形态分析.md
    │
    └→ Finish (20:30)
        ├─ Git commit + push to GitHub
        └─ Update INDEX.md with new reports
```

### Layer 5: Monitoring & Alerting (监控告警层)

#### 需要追踪的关键指标

| 类别 | 指标 | 频率 | 触发阈值 |
|------|------|------|---------|
| 账户表现 | 总资产净值 | 每日 | 单天下跌>5% 时复盘 |
| 个股跟踪 | 持仓股基本面变化 | 每周 | 财报季重点监控 |
| 市场情绪 | 换手率/成交额 | 每日 | 异常值记录分析 |
| 学习进度 | 阅读本书数 | 每月 | 未达 4 本/年需调整 |
| 健康状态 | 睡眠/运动 | 每日 | 提醒机制 |

#### 告警场景设计

```yaml
风险告警:
  - "持仓股发布利空公告" → 立即通知 + 检查卖出条件
  - "单日大盘跌幅>5%" → 次日早盘回顾是否错杀机会
  - "某只股票 PE 突破历史 90% 分位" → 考虑减仓
  
机会告警:
  - "观察池股票价格<内在价值×70%" → 进入重点关注列表
  - "优质公司出现恐慌性放量暴跌" → 人工复核是否错杀
  - "行业整体低估（历史百分位<20%）" → 加大研究力度
  
纪律告警:
  - "连续 7 天未更新学习笔记" → 提示坚持记录习惯
  - "单票仓位超过 15% 上限" → 警告风控违规
  - "季度收益率偏离目标>10%" → 检查策略是否需要微调
```
