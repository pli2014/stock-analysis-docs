## 📚 分类标准

### 📄 `daily-reports/` — 每日报告
- **内容**: 涨停板分析、资金流向、盘面点评、复盘总结
- **命名**: `YYYYMMDD-主题类型.md`
- **示例**: 
  - `20260411-涨停板.md`
  - `20260412-资金流向.md`

### 🎯 `thematic-reports/` — 专题报告  
- **内容**: 行业深度研究、技术选股策略、趋势跟踪分析
- **命名**: `YYYYMMDD-专题名称.md`
- **示例**:
  - `20260411-连续上涨.md`
  - `20260415-新能源板块.md`

### 📋 `watchlists/` — 跟踪列表
- **内容**: 核心关注标的池、持仓记录、买卖点验证
- **命名**: `YYYYMMDD-跟踪类型.md`
- **示例**:
  - `20260411-TOP20 精选.md`
  - `20260415-持仓跟踪.md`

### 📎 `attachments/` — 附件资源
- **内容**: HTML 精美版报告、K 线截图、数据导出表格
- **命名**: 与主报告保持一致
- **示例**:
  - `20260411-top20.html`
  - `20260412-k 线图.png`

## 🔄 工作流程规范

### 1️⃣ 创建分析报告
```bash
# 步骤 1: 生成 Markdown 报告
memory/YYYY-MM-DD-主题.md → 完整分析内容

# 步骤 2: 生成 HTML 精美版 (可选)
memory/YYYY-MM-DD-主题.html → Web 友好格式
```

### 2️⃣ 归类存储
```powershell
# 移动到对应月份目录
Move-Item "$source" "C:\Users\admin\stock-analysis-docs\2026\2026-04\daily-reports\"
```

### 3️⃣ Git 提交
```powershell
cd C:\Users\admin\stock-analysis-docs
git add .
git commit -m "📊 [分类] 报告标题"
git push
```
