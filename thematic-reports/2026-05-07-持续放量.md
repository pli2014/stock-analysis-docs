## 💎 核心逻辑解析

### 放量类型区分

```
✅ 健康放量特征:
   • 股价上涨 + 成交量放大 = 主动买入 (多头强势)
   • 三日累计资金流入为正 = 机构建仓确认
   • 换手率 10-25% = 筹码交换充分但不失控
   • 量比 1.5-3.0 = 温和放大可持续

❌ 危险放量特征:
   • 股价滞涨 + 成交量异常放大 = 主力派发
   • 单日换手率>40% = 游资快进快出
   • 三日资金净流出 = 利好兑现离场
   • 高位放出天量 = 阶段性顶部信号
```

## 💡 实战策略

### 放量买入法则
```
Step1: 确认是"价涨量增"而非"价滞量增"
Step2: 检查三日资金流向是否为正
Step3: 验证基本面有无支撑（业绩/政策/行业）
Step4: 设置止损位（通常为突破前一个平台低点）
Step5: 分批建仓（首次 50%，突破加 30%，回测加 20%）
```

### 退出时机
- 出现"**放量滞涨**": 成交量放大但股价不再创新高
- **换手率异常**: 单日换手>30% 且无重大利好
- **跌破关键均线**: 收盘价跌破 10 日线且次日无法收回
- **资金转负**: 三日累计由正转负
