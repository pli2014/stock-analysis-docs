## 📊 向上突破判定标准

**✅ 入选条件**:
```
[✓] 收盘价创出 N 日新高 (N≥60)
[✓] 突破当日成交量 > 前二十日均值 × 1.5
[✓] 收盘价站上五日均线和十日均线
[✓] MACD 金叉或红柱放大
[✓] 主力资金净流入 > 0 (三日累计)
[✓] 换手率 < 30% (单日)
```

**❌ 排除条件**:
```
[X] ST/*ST 股
[X] PE > 200 倍且无业绩支撑
[X] 换手率 > 50% (游资票)
[X] 突破后三日无法站稳
[X] 缩量突破 (量能不足前均值的 1.2 倍)
```

## 🔮 技术面补充分析

### 均线系统要求
- 五日线 > 十日线 > 二十日线 > 六十日线（多头排列）
- 股价站在五日均线之上
- 距离布林带上轨未超买

### 技术指标配合
- **MACD**: DIF 上穿 DEA 形成金叉，红柱持续放大
- **KDJ**: J 值未超过 90（避免超买）
- **RSI**: RSI(6) 在 50-80 区间（强势但未极端）

### 量能结构
- 突破日成交量至少是前五日均量的 1.5 倍
- 连续三日累计净流入 > 0
- 龙虎榜显示机构席位买入为主
